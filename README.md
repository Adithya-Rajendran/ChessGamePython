# Chess Game

This is a basic implementation of a Chess game.

# Setup

## Virtual Environment

Create venv

```bash
python3 -m venv .venv
```

Initialize vnev

```bash
source .venv/bin/activate
```

## Install requirements

```bash
pip3 install -r requirements.txt
```

## Runing the code

```bash
python3 main.py
```
